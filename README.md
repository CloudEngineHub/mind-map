# web思维导图的简单实现

开发中...

## 目录介绍

1.simple-mind-map

思维导图工具库。

2.web

使用`simple-mind-map`工具库，基于vue2.x、ElementUI搭建的在线思维导图。

## 开发

本地开发

```bash
git clone https://github.com/wanglin2/mind-map.git
cd simple-mind-map
npm i
npm link
cd ..
cd web
npm i
npm link simple-mind-map
npm run serve
```

打包

```bash
cd web
npm run build
```
